# Symmetry

This folder is the broad physics survey: symmetry, quantization, and the architecture of physical reality across scales.

## Contents

- `0 - overview.md` introduces the project.
- `0a - atlas of current physics.md` maps the current physics landscape.
- `1 - foundations and language.md` sets the conceptual base.
- `2` through `7` move from atomic/molecular physics through nuclear, condensed matter, particle physics, gravity, cosmology, and holography.
- `8 - cross-domain patterns.md` gathers recurring structures.
- `9 - frontiers and anomalies.md` surveys edge cases.
- `10 - references and formula sheets.md` collects references and formulas.
- `11 - boundary territories.md` marks adjacent or uncertain zones.

## Reading Path

Start with `0 - overview.md`, then read `0a - atlas of current physics.md` before diving into the domain files.
