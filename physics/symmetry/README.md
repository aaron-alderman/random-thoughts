# Symmetry

This folder is the broad physics survey: symmetry, quantization, and the architecture of physical reality across scales.

## Contents

- [0 - overview.md](0%20-%20overview.md) introduces the project.
- [0a - atlas of current physics.md](0a%20-%20atlas%20of%20current%20physics.md) maps the current physics landscape.
- [1 - foundations and language.md](1%20-%20foundations%20and%20language.md) sets the conceptual base.
- `2` through `7` move from atomic/molecular physics through nuclear, condensed matter, particle physics, gravity, cosmology, and holography.
- [8 - cross-domain patterns.md](8%20-%20cross-domain%20patterns.md) gathers recurring structures.
- [9 - frontiers and anomalies.md](9%20-%20frontiers%20and%20anomalies.md) surveys edge cases.
- [10 - references and formula sheets.md](10%20-%20references%20and%20formula%20sheets.md) collects references and formulas.
- [11 - boundary territories.md](11%20-%20boundary%20territories.md) marks adjacent or uncertain zones.

## Reading Path

Start with [0 - overview.md](0%20-%20overview.md), then read [0a - atlas of current physics.md](0a%20-%20atlas%20of%20current%20physics.md) before diving into the domain files.
