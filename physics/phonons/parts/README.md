# Phonon Parts

This folder breaks the phonon landscape into modular sections.

## Contents

- `01-conceptual-framework.md` through `10-strategic-outlook.md` form the main sequence.
- `conversion-mechanisms/` expands the conversion-mechanism section into individual mechanisms.
- `topology/` expands the topology section into foundations, topological phonons, frustration, Floquet/non-Hermitian effects, circuits, and frontier opportunities.

## Reading Path

Read the numbered files in order for the compact sequence, then use the subfolders for deeper dives.
