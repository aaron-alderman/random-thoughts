# Conversion Mechanisms - Effect Library

## Big Picture

This folder breaks the conversion-mechanisms landscape into individual effect documents. Each file keeps the same standalone format as the larger part documents, but narrows the scope to one effect at a time so design questions, bottlenecks, and opportunities are easier to track.

Together these effects describe the main ways phonons couple structured energy into voltage, cooling, spin transport, coherent phases, and useful work. Read as a set, they form the operating layer of the broader phononics program.
