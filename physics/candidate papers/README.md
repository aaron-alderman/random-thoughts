# Candidate Papers

This folder collects exploratory paper branches and research programs. Each branch is a working area rather than a finished publication set.

## Directory Map

- `spin2_3/` is the most organized active programme, with core, kernels, research tracks, papers, and a claim ledger.
- `j3_oc/` develops the J3(O) and octonionic-constraint branch.
- `e8/` explores the constraint-first E8 direction and its duality map.
- `so2_4/` compares and bridges Spin(2,3) and SO(2,4).
- `spin3_3/` holds transitional Spin(3,3) lifts and derivation branches.
- `ns/` contains Navier-Stokes/J3(O) integration notes and supporting images/documents.

## Reading Path

For the most structured branch, start with `spin2_3/README.md`. For exploratory context around the surrounding option space, read the `j3_oc/`, `e8/`, and `so2_4/` READMEs.
