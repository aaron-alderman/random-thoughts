# Candidate Papers

This folder collects exploratory paper branches and research programs. Each branch is a working area rather than a finished publication set.

## Directory Map

- [spin2_3/README.md](spin2_3/README.md) is the most organized active programme, with core, kernels, research tracks, papers, and a claim ledger.
- [j3_oc/README.md](j3_oc/README.md) develops the J3(O) and octonionic-constraint branch.
- [e8/README.md](e8/README.md) explores the constraint-first E8 direction and its duality map.
- [so2_4/README.md](so2_4/README.md) compares and bridges Spin(2,3) and SO(2,4).
- [spin3_3/README.md](spin3_3/README.md) holds transitional Spin(3,3) lifts and derivation branches.
- [ns/README.md](ns/README.md) contains Navier-Stokes/J3(O) integration notes and supporting images/documents.

## Reading Path

For the most structured branch, start with [spin2_3/README.md](spin2_3/README.md). For exploratory context around the surrounding option space, read the [j3_oc/README.md](j3_oc/README.md), [e8/README.md](e8/README.md), and [so2_4/README.md](so2_4/README.md) indexes.
