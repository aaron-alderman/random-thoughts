# Admissibility Paths

This folder compares possible paths for making the J3(O) branch admissible as physics rather than only formal structure.

## Contents

- `0 - overview and decision matrix.md` is the map and comparison table.
- `1a - Hermitian-first path.md`
- `1b - Hermitian plus rank-orbit path.md`
- `1c - Stabilizer-sensitive path.md`
- `1d - Observable-viability path.md`

## Use

Use the overview first, then read the individual path notes as alternatives with different mathematical and empirical burdens.
