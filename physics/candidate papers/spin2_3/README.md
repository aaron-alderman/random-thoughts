# Spin(2,3) Programme

This repository contains the working documents for the Spin(2,3) research programme. The structure separates core framework material, kernels, research tracks, papers, and archived snapshots.

## How To Read This Repo

- Start with [core/README.md](core/README.md), then [core/master-framework.md](core/master-framework.md) for the project spine and claim taxonomy.
- Use [core/spin23-compendium.md](core/spin23-compendium.md) as the current broad map.
- Use [kernels/README.md](kernels/README.md) for the domain-specific working documents.
- Use [CLAIM_LEDGER.md](CLAIM_LEDGER.md) to check a statement's Level 1-6 claim level.
- Use [ROADMAP.md](ROADMAP.md) for the current order of work.

## Claim Level

Research tracks may suggest new core claims, but they must pass through [CLAIM_LEDGER.md](CLAIM_LEDGER.md) before they are treated as part of the framework.
