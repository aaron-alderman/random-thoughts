# Core

This directory contains the spine of the Spin(2,3) programme.

## Reading Order

1. `master-framework.md` - the project breakdown, logical roles, and Level 1-6 scale.
2. `spin23-compendium.md` - the broad map of where the framework applies and where it does not.
3. `parent-inquiry-map.md` - ranked parent-geometry hypotheses and convergence points.
4. `ambient-reduction-scaffold.md` - the current missing-middle scaffold from parent data to observable structure.

## Claim Level

Core files should distinguish Level 1-3 inputs, Level 4 claims being actively earned, Level 5 future-work claims, and Level 6 weaknesses. Research-track material belongs in `research/` until it has passed through `CLAIM_LEDGER.md`.
