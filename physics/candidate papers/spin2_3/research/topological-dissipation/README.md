# Topological Dissipation

## Purpose

This research track holds external or semi-external models in which topology, dissipation, and quantized response interact in a mathematically sharp way. Its role is to sharpen framework language and suggest falsifiable observables without claiming that the underlying material model is itself a realized Spin(2,3) system.

## Contents

- [chern-plateaus-testbed.md](chern-plateaus-testbed.md) for the paper-level synthesis and framework-facing reading of the QWZ/Lindblad result set.

## Claim Posture

Treat this track as an analogue/testbed. The paper-local claims may be strong inside the QWZ plus Lindblad model, but any bridge to the Spin(2,3) framework remains separate work.

## Working Rule

Use this track to:

- compare how topological response changes near gap closures under controlled dephasing;
- sharpen language for Berry-curvature concentration, broadening mechanisms, and response observables;
- identify candidate measurements that distinguish dissipative fingerprints from generic smearing.

Do not use this track to:

- claim a derived Spin(2,3) material Hamiltonian;
- promote analogue results directly into [../../CLAIM_LEDGER.md](../../CLAIM_LEDGER.md);
- merge material-specific statements into the kernels without a separate bridge argument.
