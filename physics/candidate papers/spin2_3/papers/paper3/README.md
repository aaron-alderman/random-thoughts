## Claim Posture

This paper is best treated as an external condensed-matter/topological-response analogue. It sharpens the language around Berry curvature, Chern transitions, Lindblad dephasing, and experimentally testable broadening signatures, but it does not by itself establish a Spin(2,3) material realization.

Before using this paper as framework evidence, compare its claims against [../../kernels/topological.md](../../kernels/topological.md), [../../kernels/dynamics.md](../../kernels/dynamics.md), [../../kernels/consistency.md](../../kernels/consistency.md), and [../../research/topological-dissipation/README.md](../../research/topological-dissipation/README.md).
