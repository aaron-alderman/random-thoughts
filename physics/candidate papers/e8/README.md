# E8 Branch

This branch explores a constraint-first E8 program and its relation to statics, dynamics, observables, phenomenology, Higgs matching, and the surrounding duality map.

## Contents

- `0 - master.md` is the branch map.
- `statics.md`, `dynamics.md`, and `epistemics.md` develop the main kernels.
- `phenomenology.md` tracks observable consequences.
- `higgs.md` and `higgs-todorov.md` focus on Higgs/Furey/Todorov matching questions.
- `why-e8.md` explains why E8 is being considered.
- `references.md` collects key references.
- `chain/` contains short bridge notes between related structures.

## Reading Path

Start with `0 - master.md`, then read `why-e8.md` before moving into the kernel documents.
