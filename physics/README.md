# Physics

This folder gathers the physics-facing side of the repository: symmetry surveys, speculative framework notes, phonon research, candidate paper programs, and local experiments.

## Directory Map

- `symmetry/` is the broad survey of symmetry, quantization, and contemporary physics.
- `phonons/` is a focused research landscape around phonons, conversion mechanisms, topology, and strategic applications.
- `candidate papers/` contains draft research programs and paper branches.
- `experiments/` contains the local audio/response lab.
- `speculation/` holds compact speculative notes.
- `meta/` contains the physics meta-framework.
- `other/` is a parking area for material that does not yet fit an active branch.

## Start Here

Begin with `symmetry/0 - overview.md` for the broad map, then use `candidate papers/README.md` if you are tracking the active paper programs.
