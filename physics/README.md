# Physics

This folder gathers the physics-facing side of the repository: symmetry surveys, speculative framework notes, phonon research, candidate paper programs, and local experiments.

## Directory Map

- [symmetry/README.md](symmetry/README.md) is the broad survey of symmetry, quantization, and contemporary physics.
- [phonons/README.md](phonons/README.md) is a focused research landscape around phonons, conversion mechanisms, topology, and strategic applications.
- [candidate papers/README.md](candidate%20papers/README.md) contains draft research programs and paper branches.
- [experiments/README.md](experiments/README.md) contains the local audio/response lab.
- [speculation/README.md](speculation/README.md) holds compact speculative notes.
- [meta/README.md](meta/README.md) contains the physics meta-framework.
- [other/README.md](other/README.md) is a parking area for material that does not yet fit an active branch.

## Start Here

Begin with [symmetry/0 - overview.md](symmetry/0%20-%20overview.md) for the broad map, then use [candidate papers/README.md](candidate%20papers/README.md) if you are tracking the active paper programs.
