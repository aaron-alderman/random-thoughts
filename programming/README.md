# Programming

This folder applies the generation-cascade framework to software design. It treats programming paradigms as partial discoveries of deeper structural layers: identity, properties, relations, and effects.

## Start Here

- [0 - summary.md](0%20-%20summary.md) gives the full argument: how the RCHO sequence can be read as a programming model, why existing paradigms each capture only part of the structure, and what a cascade-based software system would make explicit.

## Role In The Repository

This branch is the software-facing application of the larger ontology project. It is most closely connected to `ontology/`, `relations/`, and `ai/`.
