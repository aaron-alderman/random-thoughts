# Engineering

This folder contains practical framework applications for industrial, scientific, and distributed engineering problems. The emphasis is on systems under uncertainty: how to model them, where bottlenecks appear, and what interventions become visible when structure is made explicit.

## Contents

- [science search framework.md](science%20search%20framework.md) maps an industrial engineering search space.
- [modeling framework.md](modeling%20framework.md) develops a general framework for complex distributed systems under uncertainty.
- [fertilizer.md](fertilizer.md) applies the framework to distributed fertilizer production.
- [fertilizer - western australia.md](fertilizer%20-%20western%20australia.md) localizes the fertilizer analysis to the Western Australian Wheatbelt.

## Reading Path

Start with [modeling framework.md](modeling%20framework.md) for the general method, then move to the fertilizer documents for concrete application.
