# Assets

This folder contains shared site assets used by the repository's generated pages and hosted HTML artifacts.

## Contents

- `site.css` provides shared site styling.
- `scroll.css` and `scroll.js` support `../the_scroll.html`.
- `muqarnas.css`, `muqarnas.js`, and `muqarnas.jpg` support `../muqarnas.html`.

## Notes

Keep this folder focused on reusable presentation assets. Content and prose belong in the topic folders.
