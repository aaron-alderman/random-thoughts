# Cross-Project Synthesis

This folder holds integrative material that cuts across the major branches of the repository. It gathers narrative, visual, geometric, and delivery-oriented artifacts for presenting the whole framework as one connected structure.

## Contents

- `5 - narrative.md` is the long-form narrative synthesis.
- `5D.md`, `dual-sheet.md`, `parallel-transport.html`, `phase-portrait.html`, and `symmetry-breaking.html` explore geometric and dimensional presentations.
- `narrative-crystal.md`, `gem-style-guide.md`, and `animation-targets.md` shape the visual language of the project.
- `delivery-archetypes.md`, `convergence.md`, and `third-order-influence.md` frame how the material can be communicated and inhabited.
- `cuts/` contains shorter, audience-specific slices of the same whole.

## Start Here

Use `cuts/00 - index.md` for a compact map, or `5 - narrative.md` for the expanded version.
