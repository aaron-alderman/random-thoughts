# First Cuts

This folder contains short, targeted cuts through the larger framework. Each file offers a different entry angle rather than a separate project.

## Contents

- `00 - index.md` introduces the set.
- `01 - kernel.md` gives the compressed core.
- `02 - axis-sheet.md` names the main axes.
- `03 - methodology.md` explains the method.
- `04 - technical.md` gives the technical version.
- `05 - mythic.md`, `06 - devotional.md`, and `07 - prophetic.md` shift into symbolic, religious, and exhortative registers.
- `08 - polarity.md` works through the polarity structure.

## Use

Start with `00 - index.md`, then choose the cut that matches the reader or conversation.
