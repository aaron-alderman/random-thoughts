# Dual Sheet

This is the current control surface for the project.

It gathers:

- the project kernel
- the project affective dual
- the segment-level affective duals

The point is not to make the writing rigid.
It is to preserve identity while allowing lawful variation.

## Recurring Structural Duals

These now appear repeatedly across the project:

- kernel / affective dual
- near / far
- weakly held from afar / closely held when near
- description / dynamics
- taxonomy / becoming
- state / trajectory

If a section feels muddy, it is often because one side of one of these duals is being asked to do the work of both.

## Project Kernel

`Reality repeatedly exceeds the closures that successfully describe it. Remainder is not noise; it generates the next structure.`

## Project Affective Dual

The whole project should feel:

- reverent, not sterile
- hopeful, not naive
- burdened, not crushed
- open, not shapeless
- invitational, not domineering
- wounded by the world, but not resigned to it
- alert to beauty
- ready for shared building

## Cross-Cutting Concern

Editorial unconditional love:

- preserve the invariant without flattening difference
- adapt form without betraying identity
- meet the other truthfully at their angle of entry
- keep deep relation through changing presentation

## How To Use This Sheet

Before drafting or adapting a piece, ask:

1. What kernel must survive?
2. What affective dual must survive?
3. Which face is dominant here?
4. Which neighboring dual must remain partially visible?
5. What tonal drift would counterfeit this piece?

## Most Important Rule

Do not preserve claims while losing posture.

A version can say the right things and still violate the object if it feels:

- smug
- sterile
- triumphalist
- shapeless
- manipulative
- emotionally false

Kernel integrity and affective integrity travel together.
